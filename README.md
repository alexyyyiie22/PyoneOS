# PyoneOS v1.0

A custom, self-contained terminal operating system written entirely in pure Python.

## Features
* **Shell Core Loop:** Dedicated interactive environment interface.
* **Integrated Filesystem:** Native support for creating folders (`mkdir`), creating text/scripts (`write`), reading content (`cat`), and directory mapping (`ls`).
* **Built-in Utilities:** Crash-protected basic calculation engine and system identity modules.

## How to Run
Ensure you have Python 3 installed via your environment (Termux/Linux/PC), navigate to the directory, and boot the core engine:
```bash
python3 boot.py
